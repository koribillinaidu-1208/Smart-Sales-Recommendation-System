# 🛒 Retail Intelligence System

## 📌 Overview

This project analyzes retail data to generate insights, predict sales, and recommend products.

---

## 🚀 Features

* 📊 Data Analysis (EDA)
* 🔮 Sales Forecasting (ML Model)
* 🎯 Recommendation System (Apriori)
* 📈 Interactive Dashboard (Streamlit)

---

## 🧠 Tech Stack

* Python
* Pandas, NumPy
* Matplotlib
* Scikit-learn
* MLxtend
* Streamlit
